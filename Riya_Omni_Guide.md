# If Video Does Not Look Right

After generating, type directly in Gemini Omni chat to fix:

**Looks cartoonish:**
`"Make this look like real film footage, remove all cartoon look"`

**Riya's face changed:**
`"Keep Riya's face exactly the same as the face image I gave you, do not change it"`

**Scale looks small or weak:**
`"Make the scene much bigger and more epic in scale"`

**Colors look dull or wrong:**
`"Make colors more cinematic — deep gold and teal color grade"`

**Hindi dialogue not clear:**
`"Make Riya speak the Hindi dialogue more clearly and naturally"`

**Hindi dialogue not coming properly — use ElevenLabs instead:**
If Omni is not saying Hindi dialogue correctly,
generate video without dialogue → then add Hindi voice separately using ElevenLabs.
ElevenLabs → Hindi voice → type dialogue → download → add in CapCut.

# Step 3 — Final Video Assembly in CapCut

If you made multiple clips (one per scene), join them here:

**Step 1:** Open CapCut → New Project → Import all video clips

**Step 2:** Arrange in order: Clip 1 → Clip 2 → Clip 3 → Clip 4 → Clip 5

**Step 3:** Add transitions between clips:
Use "Dissolve" or "Flash" — keep it 0.2 seconds only

**Step 4:** Add background music:
Use ElevenLabs Music v2 → generate BGM matching your scene mood
- For Vimana battle: "Epic Indian orchestral battle music, heroic, 120 BPM"
- For Krishna scene: "Soft divine flute music, peaceful spiritual, Indian classical"

**Step 5:** Add Hindi dialogue (if Omni did not generate it correctly):
ElevenLabs → Hindi female voice → type each line → download → place on timeline

**Step 6:** Color grade:
CapCut → Filters → "Cinematic" or "Golden Hour"

**Step 7:** Export:
Resolution: 1080p or 4K
Frame rate: 30fps or 60fps
